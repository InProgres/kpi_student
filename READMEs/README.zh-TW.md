# kpi_student

遊戲將我們帶入了學生生活的起點。您將以一個簡單的學生開始，在這個新城市裡沒有很多錢，這還不適合新來的遊客。
遊戲包含：

-   邏輯難題
-   戰略要素
-   困惑的情節
-   不一定是解決方案
-   財富
-   有點神秘
-   很多研究
-   以及更多

您能克服所有挑戰嗎？ KPI和他的旅館正在等待您。
